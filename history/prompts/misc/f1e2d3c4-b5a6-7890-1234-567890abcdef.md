**PROMPT_TEXT**
```
@[TerminalName: powershell, ProcessId: 12940] 
PS C:\Users\Lenovo\Downloads\physical-ai-robotics\physical-ai-robotics\textbook-frontend>npm run build
npm : File C:\Program Files\nodejs\npm.ps1 cannot be loaded because running scripts is 
disabled on this system. ...
    + CategoryInfo          : SecurityError: (:) [], PSSecurityException
    + FullyQualifiedErrorId : UnauthorizedAccess
```

**RESPONSE_TEXT**
The error `PSSecurityException` occurs because the system's PowerShell execution policy restricts running scripts like `npm.ps1`. 

**Solution:** 
Run the command using `cmd` to bypass PowerShell's script restrictions:
`cmd /c "npm run build"`

Alternatively, you could change the execution policy (requires Admin), but using `cmd /c` is safer and easier for this task.
