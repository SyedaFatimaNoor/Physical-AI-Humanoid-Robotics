# Capstone Project

## Objective
Build a simulation of a humanoid robot that can:
1. Navigate a home environment.
2. Respond to voice commands.
3. Pick up an object.

## Requirements
- Use ROS 2.
- Simulation in Gazebo or Isaac Sim.
- Video demo required.
